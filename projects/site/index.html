Simple Scrolling Parallax

Layers move at different speeds as you scroll. Use data-speed to change depth.

What this is

A minimal base page with a hero that uses layered parallax. The rest of the page scrolls normally so you can mix static and parallax content.

How to tune it

  • Change data-speed on each .parallax-layer (smaller = farther, larger = moves more).
  • Replace background-image URLs with your own photos or gradients.
  • For better performance on heavy pages, use lower-res images and limit number of layers.
