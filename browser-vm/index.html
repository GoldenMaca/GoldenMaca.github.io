What this is
A lightweight “VM-like” experience implemented as a single HTML page + JS. It keeps a virtual filesystem in memory and can “run” a tiny web app.
Why not a real VM? Real VM creation usually requires QEMU/KVM/VirtualBox + a web console bridge. You said “no browser automation required”, so this focuses on the browser UI itself.
